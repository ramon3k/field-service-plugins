How to Use

  • Place your plugin folder in server/plugins/your-plugin/
  • Make sure it has plugin.json and index.js
  • For frontend components, include them in frontend/ folder
  • Plugins auto-reload when you save changes
  • Check the browser console (F12) for errors
Note: This simulator loads plugins from the development server (port 5000). Make sure your backend is running: node server/api.cjs
